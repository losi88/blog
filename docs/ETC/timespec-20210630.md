## ERROR MESSAGE
```
Error C2011 'timespec': 'struct' type redefinition ...
```

---

## REASON

time.h already declares timespec. So it might be conflicted with other libraries.
To solve the issue, we can simply add 

```
#define HAVE_STRUCT_TIMESPEC
```
